# Ali Israr Portfolio Website

Static source code for a modern personal portfolio website that can be uploaded directly to GitHub Pages.

## Files

- `index.html` - main portfolio page
- `project.html` - project detail page template
- `project-data.js` - all 17 project titles, summaries, PDFs, image galleries, citations, and links
- `styles.css` - responsive visual design
- `script.js` - media carousel, project cards, and project content
- `assets/images/` - portrait and project images
- `assets/docs/` - CV and source PDFs
- `assets/logos/media_logos/` - media outlet logos

## Update Links And Content

Edit `project-data.js` to update project titles, summaries, PDFs, image galleries, citations, and links. Edit `script.js` for media coverage cards and shared page behavior.

## Contact Form

The contact form posts to FormSubmit:

```text
https://formsubmit.co/mail2aliisrar@gmail.com
```

On first use, FormSubmit may send a confirmation email before forwarding messages. GitHub Pages itself cannot send email, so a third-party form endpoint or a `mailto:` form is required for forwarding.

## GitHub Pages

Upload all files in this folder to a GitHub repository, then enable GitHub Pages from the repository settings. Use the repository root as the publishing source.
